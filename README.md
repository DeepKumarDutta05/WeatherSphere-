🌦️ Weather App
A responsive weather web application built using HTML, CSS, and JavaScript that provides real-time weather updates for any city using a weather API.
🚀 Features
•	Search weather by city name 
•	Real-time temperature updates 
•	Displays humidity and wind speed 
•	Weather condition icons 
•	Clean and responsive user interface 
•	Fast API data fetching using JavaScript 
🛠️ Technologies Used
•	HTML 
•	CSS 
•	JavaScript 
•	Weather API 
📸 Preview
Simple and interactive UI for checking live weather conditions instantly.
📚 What I Learned
•	API integration using Fetch API 
•	Handling asynchronous JavaScript 
•	DOM manipulation 
•	Error handling 
•	Responsive web design 
▶️ How to Run
1.	Clone the repository 
2.	Open the project folder 
3.	Run the index.html file in your browser
